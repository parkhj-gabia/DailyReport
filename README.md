# Daily Report 생성 도구

이 프로젝트는 일일 작업 기록(`dailywork.txt`)과 템플릿(`template.txt`)을 결합하여 데이터센터별 일일 업무 보고서를 자동으로 생성하는 파워쉘 스크립트 도구입니다.

## 주요 기능

- **자동화된 보고서 생성**: 반복적인 텍스트 작업을 자동화하여 보고서 작성 시간을 단축합니다.
- **데이터센터별 분류**: 하나의 작업 목록에서 '가산', '서초' 등 데이터센터별로 별도의 보고서 파일을 생성합니다.
- **서비스별 그룹화**: "하이웍스" 서비스는 별도 그룹(4번)으로 분류하고, 나머지 서비스는 일반 작업내역(2번)으로 자동 분류합니다.
- **시간순 정렬**: 야간 근무 특성을 고려하여 `00:00 ~ 08:00` 사이의 기록은 전일 업무의 연장선상으로 간주하여 가장 뒤로 정렬합니다.
- **고정 항목 및 점검 내용 자동 주입**: ARP 체크, 육안점검 완료 메일 발송 등 데이터센터별 특화된 문구를 자동으로 삽입합니다.

## 필수 파일

- `Generate-Report.ps1`: 보고서 생성 로직을 담은 핵심 파워쉘 스크립트.
- `template.txt`: 보고서의 기본 틀이 되는 템플릿 파일.
- `dailywork.txt`: 파싱할 원본 데이터 파일 (탭으로 구분된 형식).

## 사용법

파워쉘 터미널에서 아래와 같은 형식으로 실행합니다.

```powershell
.\Generate-Report.ps1 <야간근무자이름> <입력파일명>
```

> [!IMPORTANT]
> **스크립트 실행 오류 발생 시 (보안 오류, PSSecurityException)**
> 
> 만약 "디지털 서명되지 않았습니다" 또는 "UnauthorizedAccess" 오류가 발생한다면, 윈도우 보안 정책에 의해 차단된 상태입니다. 다음 명령어를 파워쉘에서 실행하여 차단을 해제하세요.
> 
> ```powershell
> Unblock-File -Path .\Generate-Report.ps1
> ```
> 
> 그래도 실행되지 않는 경우, 임시로 보안 정책을 우회하여 실행할 수 있습니다.
> ```powershell
> powershell -ExecutionPolicy Bypass -File .\Generate-Report.ps1 <야간근무자이름> <입력파일명>
> ```

### 실행 예시

```powershell
.\Generate-Report.ps1 Benedict dailywork.txt
```

### 출력 결과

실행 후 다음과 같은 형식의 결과 파일들이 현재 디렉토리에 생성됩니다.
- `dailyreport_가산_20260310.txt`
- `dailyreport_서초_20260310.txt`

> [!WARNING]
> 주의) 작업자의 이름은 수정해서 사용해야 합니다.

## 데이터 입력 형식 (dailywork.txt)

`dailywork.txt` 내용은 **"운영포털 > 작업 > 관제업무보고 > 기술지원 업무보고"**에 있는 내용을 그대로 복사하여 저장합니다.

데이터는 탭(`\t`)으로 구분된 8개 이상의 열로 구성되어야 합니다.

| 순서 | 항목 | 설명 |
| :--- | :--- | :--- |
| 1 | 일시 | `YYYY-MM-DD HH:MM` 형식 |
| 2 | 근무자 | 작업자 이름 |
| 3 | 데이터 센터 | 가산, 서초 등 (출력 파일 분리 기준) |
| 4 | 그룹 | 분류 그룹 (스크립트 내 필터링 기준) |
| 5 | 서비스 | 하이웍스 여부에 따라 2번/4번 그룹으로 나뉨 |
| 6 | 회원아이디 | 고객/회원 식별 정보 |
| 7 | 아이피 | 관련 IP 주소 |
| 8 | 처리내용 | 구체적인 작업 내용 |

## 변환 규칙 요약

- **날짜**: 실행일 기준으로 **어제 날짜**를 보고서 날짜로 설정합니다.
- **작업내역(2번)**: 하이웍스를 제외한 모든 서비스. `[HH:MM] 아이디 / IP - 내용` 형식.
- **가비아시스템지원(4번)**: 하이웍스 서비스 전용. 고정 VOC 문구 및 ARP 체크 정보가 포함됩니다.
- **육안점검(6번)**: 가산/서초 데이터센터별로 지정된 메일 발송 완료 문구를 삽입합니다.
